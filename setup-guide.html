Solução de Problemas

Python não encontrado no PATH

Certifique-se de que o Python foi adicionado ao PATH durante a instalação. No Windows, reinstale o Python marcando a opção "Add Python to PATH".

# Windows: verificar PATH
$env:PATH -split ';' | Where-Object { $_ -like '*Python*' }
Erro de permissão ao ativar venv no PowerShell

Execute o PowerShell como Administrador e altere a política de execução:

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
gcloud auth falha com erro de browser

Use a flag --no-launch-browser e copie a URL manualmente:

gcloud auth login --no-launch-browser
ADK não reconhecido após instalação

Verifique se o ambiente virtual está ativado e reinstale o ADK:

# Verificar ambiente ativo
pip list | grep google-adk

# Reinstalar se necessário
pip install --upgrade google-adk
